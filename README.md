# dicorato-luthier-site

Estado: Activo

Sitio oficial del taller Di Corato Luthier. Presenta instrumentos de autor, servicios de reparación y canal de contacto directo.

---

## Objetivo

- Exponer modelos de instrumentos con claridad
- Documentar trabajos y procesos del taller
- Centralizar consultas de clientes
- Funcionar como presencia digital estable del taller

---

## Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS 3
- pnpm
- Export estático (output: "export")

---

## Requisitos

- Node.js 18+
- pnpm 9+

---
